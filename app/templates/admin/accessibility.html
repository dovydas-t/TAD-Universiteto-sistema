Accessibility Statement

This Accessibility Statement (the Statement) was prepared on April 25, 2024.

This Statement applies to content published on the main domain ‘en.ktu.edu’ and its subdomains ‘admissions.ktu.edu’ and ‘students.ktu.edu’ (the Website). The Website is managed by Kaunas University of Technology.

Compliance status

The Website is partially compliant with technical standard EN 301 549 v.3.2.1 and the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA. See more information in the section Limitations and alternatives.

Compliance status is based on the results of Accessibility Evaluation. The detailed results are listed in the Accessibility Conformance Report.

Accessibility Features

In addition to many other accessibility features, when using the Website, you should be able to:

  • Zoom in up to 200% without any issues
  • Easily navigate the Website using only your keyboard.
  • Seamlessly browse the Website with modern screen readers and speech recognition software on your computer or phone.

Feedback

We welcome your feedback on the accessibility of this Website. Please let us know if you encounter accessibility barriers by:

  • Email: prieinamumas@ktu.lt
  • Feedback form (please avoid sharing sensitive details in the form, such as personal financial information, health-related details, or any other sensitive topics).

Preparation of this Statement

The Statement is based on a review of a representative sample of web pages conducted by:

  • Sighted digital accessibility expert. The evaluation was done by combining manual and automated testing.
  • Digital accessibility expert with a visual impairment. The evaluation was done using assistive technologies.

Compatibility with Assistive Technologies

The Website is designed to be largely compatible with the following commonly used assistive technologies:

  • “Jaws” and “NVDA” screen readers and Chrome on Windows.
  • “Talkback” screen reader and Chrome on Android.
  • “VoiceOver” screen reader and Safari on iOS.

Technical specifications

The accessibility of the Website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Limitations and alternatives

When using the Website with assistive technologies, you may encounter certain limitations. Below, we outline these known issues and offer potential solutions:

  • Keyboard Accessibility. Some interactive elements, such as sliders and dropdown menus, cannot currently be navigated using a keyboard alone.
  • Visible Focus Indicators. The Website does not consistently show visible focus indicators on some elements, which are crucial for users navigating via keyboard.
  • Descriptive Link Text. Many links on the Website lack clear descriptions, hindering navigation for both screen reader users and those who need clear context to navigate effectively.
  • Color Contrast. Some text and background combinations do not meet the minimum color contrast ratio, which can make text difficult to read for users with visual impairments.
  • Video Accessibility Options. Some videos lack audio descriptions, transcripts, and captions, which are essential for users with visual impairments or cognitive disabilities to fully understand and process the content.
  • Text Alternatives for Images. Some images on the Website may lack text alternatives or may have inaccurate descriptions. We actively monitor user feedback and typically address and correct any reported issues within 20 working days.
  • Accessibility of Archived Documents. Some archived documents may not be compatible with current assistive technologies due to the use of outdated technologies. Upon request, we will convert these documents to accessible formats within 20 working days.

The limitations are listed in more detail in the Accessibility Conformance Report.

We are committed to making our website accessible to all users and recognize the need for improvements in several areas. We are actively working to enhance these features and plan to implement significant upgrades in the coming months. We value your patience and welcome your feedback as we work to improve your experience.
